# Into The Details

An open source e-commerce platform built with Next.js 14, Supabase, and Stripe.

## Features
- Framework: Next.js 14 App Router
- Auth: Supabase Auth (SSR)
- Database: Supabase / Drizzle ORM
- Payments: Stripe
- Styling: Tailwind CSS / Shadcn UI
- Components: Radix UI

## Getting Started
1. Clone the repo
2. Install dependencies: `npm install`
3. Setup `.env`
4. Run dev: `npm run dev`
